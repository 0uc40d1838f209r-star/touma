# 画像の差し替え方（Japanese Matcha Store 39）

現在このフォルダの画像は、**ライセンスが明確なフリー素材**（Wikimedia Commons）を
使用しています。自社で撮影した写真に差し替えたい場合は、
**同じファイル名で上書き**するだけでサイトに反映されます。

## 差し替え手順
1. VS Code のエクスプローラーでこの `assets` フォルダを開く
2. 自社の写真をドラッグ＆ドロップ
3. 下の表のファイル名にリネーム（既存を上書き）
4. `git add` → `git commit` → `git push` で公開サイトに反映

## 注意
- ファイル名は**半角英数・小文字**のまま変えないでください
- 画像は**1枚500KB程度**まで圧縮すると表示が速くなります
- **自社写真に差し替えた画像は、フッターの「写真クレジット」から
  その行を削除してください**（フリー素材のクレジット表記が不要になるため）

## 動画背景にしたい場合
現在は写真4枚のスライドショーですが、動画背景も可能です。
`hero.mp4`（10MB以下推奨）をご用意いただければ組み込みます。
