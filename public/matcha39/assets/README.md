# 画像・動画の入れ方（Japanese Matcha 39）

このフォルダに**下記のファイル名で**画像・動画を置くと、サイトが自動的にその素材を使います。
**ファイルが無い箇所は仮のフリー画像が表示される**ので、1枚ずつ順番に入れて大丈夫です。
サイトが途中で壊れることはありません。

## 入れ方
VS Code の左サイドバー（エクスプローラー）で、この `assets` フォルダに
写真・動画を**ドラッグ＆ドロップ**するだけです。
その後、ファイル名を下の表のとおりに変更してください（右クリック →「名前の変更」）。

## 推奨サイズ・形式
- **画像**: `.jpg` 推奨。横1600〜2000px程度。1枚 **500KB以下**が理想
- **動画**: `.mp4`（H.264）。**10MB以下**を強く推奨（重いとスマホで表示が遅くなります）
- Instagramからダウンロードした素材より、**撮影時の元ファイル**の方が高画質できれいです

## 注意
- ファイル名は**半角英数字**のまま変えないでください（日本語名だと表示されません）
- 大文字小文字も区別されます（`Hero.mp4` は不可、`hero.mp4` が正しい）
- 差し替え後は `git add` → `git commit` → `git push` すると公開サイトに反映されます
